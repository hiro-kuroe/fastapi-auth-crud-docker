# 🔎 進め方について

ご連絡後、まずは **分析フェーズ（原因の層の特定と構造整理）** をご提案します。

分析フェーズでは：

- 事象の層分解
- 原因の特定
- 修正方針の提示
- 再発防止観点の整理

を成果物としてご提示します。

※修正作業は、原因確定後に別途ご提案となります。
